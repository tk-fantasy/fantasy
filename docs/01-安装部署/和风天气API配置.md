# 和风天气 API 配置

这篇讲怎么配天气服务。Aether 用和风天气，配好后助手能告诉你今天冷不冷、要不要带伞，侧边栏还会显示实时天气。

## 怎么到配置页

天气配置在 **`/advanced` 高级设置**页里（聊天框没有专门斜杠命令，从侧边栏点「高级」进入，或浏览器访问 `http://localhost:5173/advanced`）。页面里有一块「天气 API（和风天气）」分区。

## 拿天气服务凭证

### 1. 注册账号

打开 https://www.qweather.com 注册开发者账号。

### 2. 创建项目

进控制台 → **项目管理** → **创建项目**，选免费订阅。

### 3. 拿到凭证

项目创建好后，详情页能看到四个东西：

| 凭证 | 说明 | 示例 |
|------|------|------|
| Host | API 主机地址 | `xxx.re.qweatherapi.com` |
| Key ID (kid) | 公开 ID | `KCWHAKR6HC` |
| 订阅 ID (sub) | 订阅标识 | `2G2D55DC98` |
| 私钥 (private_key) | Ed25519 私钥（base64） | `MC4CAQAwBQYDK2VwBCIE...` |

> 和风天气用 **JWT (Ed25519 签名)** 认证，不是普通 API Key。Aether 会用私钥自动签发 15 分钟有效的 JWT，你只要把四个凭证填对就行。

### 4. 填入配置

在 `/advanced` 页的天气分区填这四个字段：

- **Host**：和风控制台给的 API 主机地址
- **Key ID (kid)**
- **Subscriber (sub)**
- **Private Key**：私钥（密码框，已配置过的话留空表示不修改）

填完点页面底部的 **「保存所有配置」**。

## 设你家位置

天气查询需要你的位置。去 **`/settings` 设置页**（侧边栏「设置」）填：

- 家庭名字
- 主人称呼（AI 聊天时用）
- 省 / 市 / 区县（三级联动下拉，选省才解锁市，选市才解锁区）

> 直辖市（北京/上海/天津/重庆）省市填一样。

Aether 查天气用省市定位。也可以让 Aether 通过 IP 自动定位（`/api/weather/locate`）。

## 天气数据缓存

Aether 不会每次都问和风（免费订阅有次数限制），结果本地缓存 **15 分钟**。同一地点短时间重复查用缓存，不浪费额度。

缓存内容：
- 当前天气：温度、湿度、是否下雨、风力
- 生活指数：运动、洗车、穿衣、感冒、紫外线、开空调、晾晒

> 自动化规则评估时也会用天气做条件，天气缓存 60 秒（比前端短，避免规则用过期数据）。

## 侧边栏天气组件

侧边栏底部有个天气小部件，显示：

| 信息 | 说明 |
|------|------|
| 天气图标 | 根据天气自动匹配（可点开换 Emoji） |
| 当前温度 | 摄氏度 |
| 体感温度 | 实际感受温度 |
| 天气描述 | 晴、多云、小雨等 |
| 相对湿度 | 百分比 |
| 风向风速 | 如「东南风 3级」 |
| 能见度 | 公里 |
| 紫外线 | 强度等级 |
| 生活指数 | 运动/洗车/穿衣/感冒/紫外线/开空调/晾晒 |

### 夜间自动变暗

19:00~05:59 算夜间，图标和背景色自动调暗，不刺眼。

### 换天气图标

点天气图标，从 Emoji 里挑一个喜欢的，偏好存起来下次还在。

### 刷新频率

每 15 分钟自动刷新一次。

## 天气相关的 API

| 接口 | 用途 |
|------|------|
| `GET /api/weather` | 取当前天气（带缓存） |
| `GET /api/weather/locate` | IP 自动定位 |
| `GET /api/weather/city` | 查城市信息 |
| `GET /api/weather/indices` | 查生活指数 |
| `GET /api/weather/config` | 取天气配置（不返回私钥） |
| `POST /api/weather/config` | 保存天气配置 |

## 出问题了怎么办

### 凭证不对

**现象**：查天气返回 `获取天气失败` 或签名错误

- 确认四个凭证属于同一个项目
- 确认私钥没多余换行或字符
- 检查订阅有没有过期（免费订阅每天有调用上限）

### 查不到你家天气

**现象**：返回 `找不到地点`

- 检查 `/settings` 页省市填对了没
- 直辖市省市填同一个

### 天气不更新

**现象**：一直显示同样温度

- 缓存 15 分钟，等一等会自动刷新
- 想立刻刷新就重启后端
- 看 `logs/app.log` 有没有报错

### 连不上天气服务器

**现象**：请求超时或域名解析失败

- 确认 Host 填的是控制台给的实际地址（格式如 `xxx.re.qweatherapi.com`）
- 确认网络能访问和风天气

搞定了！天气配好后，跟 Aether 说"今天天气怎么样"，它会告诉你该穿什么。
